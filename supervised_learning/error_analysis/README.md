\# Error Analysis



This project covers techniques for analyzing model errors, including

confusion matrices, precision, recall, specificity, F1 score, and the

bias-variance tradeoff.



\## Learning Objectives



\- What is a confusion matrix?

\- What is type I / type II error?

\- What is sensitivity, specificity, precision, recall?

\- What is an F1 score?

\- What is bias, variance, irreducible error, Bayes error?



\## Requirements



\- Python 3.5, numpy 1.15

\- Files start with `#!/usr/bin/env python3`

\- Files must be executable

\- Code follows `pycodestyle` (2.4)

\- All modules/functions documented
