\# Decision Tree



This project covers building a decision tree from scratch, including node

and leaf structures, computing tree depth, counting nodes, and traversing

the tree to make predictions.



\## Learning Objectives



\- What is a decision tree?

\- What is a root node? a leaf? an internal node?

\- What is the depth of a tree?

\- How to compute the depth of a decision tree

\- How to count the number of nodes/leaves in a decision tree

\- What are the left-child and right-child of a node?

\- What is a random forest?

\- What is the difference between a decision tree and a random forest?



\## Requirements



\- Python 3.5, numpy 1.15

\- Files start with `#!/usr/bin/env python3`

\- Files must be executable

\- Code follows `pycodestyle` (2.4)

\- All modules/classes/functions documented
